Posted on 2019-12-08 Edited on 2019-12-09 In Python Crawl Views: Valine:

本文介绍了正则表达式的语法、语义.

|符号|描述|
|:—:|:—:|
|\|转义字符|
|^|匹配输入字符首|
|$|匹配输入字符尾|
| | 匹配前面的表达式任意 n 次。例如：zo 能匹配“z”，也能匹配“zo”以及“zoo”。等价于{0,}|
|+|匹配前面表达式 1 次或多次 等价于{1,}|
|?|匹配前的表达式 1 次或 0 次 等价于{0,1}|
|?|也用于非贪婪模式匹配.正则表达式默认贪婪模式|
|{n}|n 是一个非负整数,匹配确定的 n 次|
|{n,}|n 是一个非负整数,表示至少匹配 n 次 o{0,}等价于o*,o{1,}等价于o+|
|{n,m}|表示匹配 n 到 m 次|
|.|表示任意匹配|
|a|b|表示匹配 a 字符或者 b 字符|
|[xyz]|匹配所包含的任意一个字符|
|xyz|匹配未包含的任意字符|
|[a-z]|匹配指定范围的任意字符|
|a-z|匹配不在范围的任意字符|
|\\b|匹配一个单词的边界.例如，“er\b”可以匹配“never”中的“er”,但不能匹配“verb”中的“er”;“\b1_”可以匹配“1_23”中的“1_”,但不能匹配“21_3”中的“1_”,|
|\\B|与\\b相反,匹配非单词边界|
|\\cx||
|\\d|等价于 [0-9]|
|\\D|等价于0-9|
|\\f|匹配一个换页符|
|\\n|匹配换行符|
|\\r|匹配回车符|
|\\s|匹配任何不可见字符|
|\\S|匹配任何可见字符|
|\\t|匹配制表符|
|\\v|匹配垂直指标|
|\\w|匹配包括下划线(“_”)的任意字符|
|\\W|匹配任何非单词字符|
